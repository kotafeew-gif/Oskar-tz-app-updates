================================================
 TZ Oscar-Print - Build Instructions
================================================

REQUIREMENTS:
  - Node.js 18 or newer  (https://nodejs.org)
  - Internet connection for first build

HOW TO BUILD EXE:
  1. Double-click build.bat
  2. Wait for the build to complete
  3. The installer will appear in: electron-app\release\

ICON:
  - Place your icon.ico file in the electron-app\ folder
    before running build.bat.
  - If icon.ico is missing, the build still works
    but the app will use the default Electron icon.
  - You can convert any PNG to ICO at: https://convertico.com

OUTPUT:
  - electron-app\release\TZ Oscar-Print Setup 1.0.0.exe

================================================
